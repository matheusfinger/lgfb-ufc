Sobre o LGFB

O Laboratório de Genômica Funcional e Bioinformática (LGFB) dedica-se ao estudo integrado de dados ômicos...

Nossa missão é aplicar técnicas computacionais avançadas para resolver problemas biológicos complexos.

[ Foto do Laboratório / Equipamentos ]

Linhas de Pesquisa

Genômica

Análise de sequenciamento de nova geração (NGS) e montagem de genomas.

Bioinformática

Desenvolvimento de pipelines, modelagem de dados e análise estrutural de proteínas.

Análises bioquímicas

Integração de dados de transcriptômica e proteômica para entender vias metabólicas.
